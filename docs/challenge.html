ANNOTED: Taming protein beasts through visualization

This repository implements an interactive visualization intended to address the requirements of the Bio+MedVis Challenge @ IEEE VIS 2022: Taming protein beasts through visualization (http://biovis.net/2022/biovisChallenges_vis/).

The visualization displays a protein structure with colored residues. Some of the residues of the structure are annotated. Annotated residues for selected annotations are marked by a circle.

Hovering or clicking an annotated residue displays detail about the residue annotation in the right panel.

Interactions:

  • Turn autorotation on or off using the checkbox.
  • Rotate the figure by dragging over the display area.
  • Inspect an annotated residue by hovering over the circle marker for the residue.
  • Focus on a residue by clicking the circle marker for the residue.
  • Examine proximal structure to the focus by adjusting the radius slider.
  • Select annotation types of interest using the annotation checkboxes.
  • Fit the figure to the view port using the fit button.
  • Use the pathogenic checkbox (if present) to show only pathogenic annotations.
